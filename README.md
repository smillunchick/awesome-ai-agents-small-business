## Agent Frameworks

Open-source and commercial frameworks for building AI agents.

- [CrewAI](https://github.com/crewAIInc/crewAI) — Multi-agent orchestration framework. Build teams of AI agents that collaborate on complex tasks. Python-based.
- [AutoGen](https://github.com/microsoft/autogen) — Microsoft's framework for building multi-agent conversational systems. Supports complex agent workflows.
- [LangGraph](https://github.com/langchain-ai/langgraph) — Framework for building stateful, multi-step agent applications. Part of the LangChain ecosystem.
- [Semantic Kernel](https://github.com/microsoft/semantic-kernel) — Microsoft's SDK for integrating LLMs into applications. C#, Python, and Java support.
- [Haystack](https://github.com/deepset-ai/haystack) — End-to-end NLP framework by deepset. Build production-ready LLM applications and agents.
- [Flowise](https://github.com/FlowiseAI/Flowise) — Drag-and-drop UI for building LLM flows and agents. Low-code approach.
- [Langflow](https://github.com/langflow-ai/langflow) — Visual framework for building multi-agent applications. Built on top of LangChain.
- [OpenClaw](https://github.com/open-claw) — Lightweight agent runtime with built-in memory management and tool access. Popular for self-hosted deployments.
- [Phidata](https://github.com/phidatahq/phidata) — Toolkit for building AI assistants with memory, knowledge, and tools. Production-focused.
- [Composio](https://github.com/ComposioHQ/composio) — Integration platform for AI agents. 150+ tools and integrations out of the box.

## Phone & Receptionist Agents

AI systems that answer phone calls, qualify leads, and book appointments.

- [Bland.ai](https://www.bland.ai/) — AI phone agent platform. Build agents that make and receive calls with natural-sounding voices. Per-minute pricing.
- [Synthflow](https://synthflow.ai/) — No-code AI voice agent builder. Handles inbound/outbound calls with calendar integration.
- [Vapi](https://vapi.ai/) — Developer platform for building voice AI agents. API-first approach with real-time conversation.
- [Retell AI](https://www.retell.ai/) — Build human-like AI voice agents. Supports custom LLMs and low-latency conversations.
- [Air AI](https://www.air.ai/) — Autonomous AI agent for phone sales and customer service. Handles full conversations.
- [Goodcall](https://www.goodcall.com/) — AI phone agent for small businesses. Answers calls, takes messages, books appointments.
- [Smith.ai](https://smith.ai/) — Virtual receptionist service with AI-augmented call handling. Hybrid human + AI approach.
- [Ruby Receptionist](https://www.ruby.com/) — Live virtual receptionist with AI features. Premium human-first approach.

- [Drift](https://www.drift.com/) — Conversational AI for B2B. Chatbots that qualify leads and book meetings.
- [Tidio](https://www.tidio.com/) — AI chatbot + live chat for e-commerce and small business. Lyro AI agent included.
- [Freshdesk Freddy AI](https://www.freshworks.com/freshdesk/) — AI-powered customer support agent. Auto-resolves tickets and suggests responses.
- [Chatwoot](https://github.com/chatwoot/chatwoot) — Open-source customer engagement platform. Self-hostable with AI integrations.
- [Botpress](https://github.com/botpress/botpress) — Open-source chatbot builder with AI capabilities. Visual flow editor.
- [Voiceflow](https://www.voiceflow.com/) — Build AI agents for customer support across chat, voice, and web. No-code builder.

## No-Code Agent Builders

Build AI automations without writing code.

- [Make (Integromat)](https://www.make.com/) — Visual automation platform. Connect apps and build AI workflows with drag-and-drop.
- [n8n](https://github.com/n8n-io/n8n) — Open-source workflow automation. Self-hostable with AI agent capabilities.
- [Zapier](https://zapier.com/) — Connect 6,000+ apps with automated workflows. AI features for natural language automation.
- [Activepieces](https://github.com/activepieces/activepieces) — Open-source no-code automation. Growing alternative to Zapier.
- [Relevance AI](https://relevanceai.com/) — Build and deploy AI agents and tools without code. Agent-first approach.
- [Stack AI](https://www.stack-ai.com/) — No-code platform for building AI applications and agents.
- [MindStudio](https://www.mindstudio.ai/) — Build AI applications and agents with a visual builder. Deploy to web, Slack, or API.
